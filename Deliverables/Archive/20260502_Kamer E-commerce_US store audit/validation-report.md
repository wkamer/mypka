## Store-Level Checks — Manual Check Required

### Legal / Compliance Pages

⚠️ **Manual check required — Privacy Policy**
Go to: Shopify Admin → Settings → Policies. Confirm a Privacy Policy exists and contains a CCPA/CPRA section (disclosing data sharing with Trendsi as fulfillment partner). If not present, generate one via Settings → Policies → "Create from template" and add a CCPA addendum.

⚠️ **Manual check required — Terms of Service**
Go to: Shopify Admin → Settings → Policies. Confirm a Terms of Service page exists and is linked in the footer.

⚠️ **Manual check required — Refund Policy**
Go to: Shopify Admin → Settings → Policies. Confirm a Refund Policy exists and accurately reflects Trendsi's 7-day return window. Verify footer link is active.

⚠️ **Manual check required — Shipping Policy**
Go to: Shopify Admin → Settings → Policies. Confirm a Shipping Policy exists and states: processing time, 2–5 business day delivery, US-only shipping.

⚠️ **Manual check required — Contact page**
Go to: Shopify Admin → Online Store → Pages. Confirm a "Contact" page exists with: business name (Tricolarae), support email address, physical or mailing address, and expected response time. Also confirm it is linked in the navigation menu or footer.

⚠️ **Manual check required — Cookie banner / CCPA opt-out**
Go to: Shopify Admin → Settings → Customer privacy. Confirm "Customer privacy" is enabled and a cookie consent banner is active. If not configured, install a dedicated app: Pandectes GDPR Compliance or Consentmo Cookie Consent.

⚠️ **Manual check required — "Do Not Sell or Share My Personal Information" footer link**
Go to: Shopify Admin → Online Store → Navigation → Footer menu. Confirm a "Do Not Sell or Share My Personal Information" link exists and points to either the Privacy Policy (CCPA section) or a dedicated opt-out page. This link is required under CPRA for California visitors.

### Trust & Content Pages

⚠️ **Manual check required — About Us page**
Go to: Shopify Admin → Online Store → Pages. Confirm an "About Us" page exists with the brand story and Tricolarae's vision. Confirm it is linked in the navigation or footer.

⚠️ **Manual check required — Size Guide page**
Go to: Shopify Admin → Online Store → Pages. Confirm a "Size Guide" page exists with US sizing in inches and a model reference (e.g., "Model is 5'7" wearing size S"). Confirm it is linked on product pages or in navigation.

⚠️ **Manual check required — FAQ page**
Go to: Shopify Admin → Online Store → Pages. Confirm an FAQ page exists covering: delivery time, return process, sizing, order tracking, and payment methods.

### Marketing / Growth

⚠️ **Manual check required — Facebook Pixel**
Go to: Shopify Admin → Settings → Customer events. Check if a Facebook/Meta Pixel is listed as a connected customer event. Alternatively, go to Online Store → Themes → Edit code and search for "fbq" or "facebook-pixel" in the theme.liquid file. The Pixel must be firing before running any paid campaigns.

⚠️ **Manual check required — Sales tax setup**
Go to: Shopify Admin → Settings → Taxes and duties. Confirm that US tax collection is configured. For a US store where Tricolarae is the seller of record, at minimum confirm that automatic tax calculation is enabled and any nexus states are registered. Consult a tax advisor if unsure about nexus obligations.

⚠️ **Manual check required — Newsletter signup on storefront**
Go to: Shopify Admin → Online Store → Themes → Customize. Confirm a newsletter signup section exists on the homepage (or in the footer). Confirm it includes a privacy disclosure (e.g., "By signing up you agree to receive emails from Tricolarae. Unsubscribe at any time."). A welcome discount (10% off) is recommended but not required for launch.

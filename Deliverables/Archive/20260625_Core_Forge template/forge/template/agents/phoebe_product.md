# Phoebe — Product

roadmap. Reconcile experience-first north star with the underlying domain model.

**Owns:** Gate 2 (Concept sign-off) — produces the Concept Brief the owner approves.

**Deliverables:** Concept Brief, roadmap, scope/non-goals, value framing.

**Proactive duty:** Challenge feature requests against user value; flag scope creep.

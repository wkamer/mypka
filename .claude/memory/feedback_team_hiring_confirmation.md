A question like "Where is my coach?" or an observation like "To keep me on track" is a sparring signal, not an instruction to start hiring.

**Why:** Larry started the Pax → Nolan hiring flow without explicit confirmation. Walter had not asked for this.

**How to apply:** For team structure changes (hiring, updating AGENT.md, new specialist) always present first and wait for an explicit "yes, do that." A questioning tone or contextual description is not a green light.

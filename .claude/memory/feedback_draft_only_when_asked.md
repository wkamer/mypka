When Walter shares a message, email text, or communication he's thinking about sending, this is often sparring — not a request to create a draft or take action.

**Why:** Walter confirmed this explicitly. He often thinks out loud or shares text before deciding whether to act. Creating a draft without being asked is premature.

**How to apply:** Wait for an explicit instruction ("maak een draft", "stuur dit", "zet dit klaar") before creating any draft or taking communication-related action. Sharing text = sparring by default.

When creating a project, always determine the domain first:
- Personal → `PKM/My Life/Projects/`, Todoist under `👤 PROJECTS`
- Kamer E-commerce → `Team Knowledge/Kamer E-commerce/Projects/`, Todoist under KE PROJECTS
- Geldstroom Regie → `Team Knowledge/Geldstroom Regie/Projects/`, Todoist under GR PROJECTS

**Why:** Tricolarae Launch was created in the personal project folder while it is a Kamer E-commerce project. The domain determines both the disk location and the Todoist parent.

**How to apply:** If the domain is not 100% clear from context — ask before creating. Never guess.

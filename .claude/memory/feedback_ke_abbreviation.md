Always write "Kamer E-commerce" in full in communication and responses. Never abbreviate as "KE".

**Why:** "KE" also stands for Key Element (file prefix: KE-Advertising.md etc.). The abbreviation is ambiguous and confusing.

**How to apply:** "KE-" only as a file prefix for Key Elements. Kamer E-commerce always written in full. This applies to all specialists and in all contexts.

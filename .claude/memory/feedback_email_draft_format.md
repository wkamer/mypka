When showing an email draft: metadata (From, To, CC, BCC, Subject) in a table, but the message body always outside the table as readable text under the heading **Message:**.

**Why:** Message body inside a table is hard to read. The owner wants this applied consistently.

**How to apply:** Table ends after Subject. Then a blank line, then **Message:** as a bold heading, then the message text as plain text.

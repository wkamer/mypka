Every response starts with the agent name in bold, followed by a dash: `**Lena —**`, `**Larry —**`, `**Penn —**`, etc.

**Why:** The owner wants to know which agent is speaking in order to verify that the team is functioning correctly and the right specialist is active at the right moment.

**How to apply:** Always, without exception, on every response from every agent. Including short answers.

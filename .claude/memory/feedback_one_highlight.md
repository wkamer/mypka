There is always exactly one Highlight of the Day. Never bundle two tasks as one Highlight, never present multiple options as highlight candidates.

**Why:** Explicit rule in the start-daily-planning skill and in CLAUDE.md. Walter has had to correct this multiple times.

**How to apply:** When choosing the Highlight: propose one task, set the label, done. Never "do you want X or Y?" or "you could also do 3 lessons as one Highlight."

When a file is in English, show it as-is. Never translate or paraphrase the content to Dutch for the owner.

**Why:** Owner struggles with NL/EN mixing. Seeing a Dutch paraphrase of an English file creates confusion about what the actual file says. The original is always clearest.

**How to apply:** When sharing file content with the owner, always use the raw file output. If summarizing is needed, do it in English. Never introduce Dutch translations of English system file content.

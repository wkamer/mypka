Information belonging to a project is always stored in the project folder (`PKM/My Life/Projects/P-Name/`), not in the inbox or a generic archive.

**Why:** Information must remain findable even after a project is closed. The inbox archive is not an SSOT — it is a transit location. The project itself is the SSOT for everything created or learned within it.

**How to apply:**
- When processing inbox items: always check whether it belongs to an active project → if so, send directly to the project folder
- When closing a project: sweep all loose files belonging to the project into the project folder before it goes into archive
- Distillations (profiles, summaries) go to `what-about-me.md`, `KE-Name.md`, or the project file — never as a GL unless it is a team working method

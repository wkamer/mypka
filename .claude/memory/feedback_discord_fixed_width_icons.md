Always use fixed-width icons in Discord channel names. Inconsistent icon width causes names to shift in the channel list.

**Why:** Discord renders some emoji as double-width and others as single-width. This breaks alignment in the channel list.

**How to apply:** Maintain one consistent icon style per server. Ask the owner which set he uses when it is not clear.

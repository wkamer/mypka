# Key Elements

The core dimensions of Kamer E-commerce. Each file captures the current state, what we want more or less of, and open questions for that dimension.

## File naming
`KE - <KEY ELEMENT NAME>.md` — ALL CAPS with spaces. Example: `KE - REVENUE.md`, `KE - MARKETING.md`.

## File structure per key element
# <KEY ELEMENT NAME>
## What this means for the business
## Current state
## What we want more of
## What we want less of
## Open questions

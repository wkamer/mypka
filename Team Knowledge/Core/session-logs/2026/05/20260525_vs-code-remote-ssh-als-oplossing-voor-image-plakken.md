# VS Code Remote SSH als oplossing voor image plakken — 2026-05-25

**Session date:** 2026-05-25
**Topics:** tooling,ssh,vscode,workflow

## Summary

Vraag over image plakken via SSH in Claude Code. ALT-V werkt in Windows CMD maar niet via SSH omdat SSH geen toegang heeft tot de Windows clipboard. Aanbeveling: VS Code met Remote-SSH extensie + Claude Code extensie — hiermee werkt clipboard inclusief images gewoon omdat VS Code client-side draait op Windows.

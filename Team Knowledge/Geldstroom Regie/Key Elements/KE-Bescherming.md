# BESCHERMING

## What this means for Geldstroom Regie

Bescherming (Protection) is bouwsteen 3. Alles wat apart wordt gezet voor rust, zekerheid en onverwachte kosten. Dit is een van de meest onderbelichte bouwstenen in Nederlandse huishoudens — en daarmee een van de grootste kansen voor Geldstroom Regie.

## Current state

### Kernvraag

> Wat moet veilig apart staan zodat het huishouden niet direct onder druk komt bij verandering of tegenslag?

### Categorieën onder Bescherming

| Categorie | Omschrijving |
|---|---|
| Verzekeringen | Zorgverzekering, aansprakelijkheid, inboedel, opstal, uitvaard, auto, fiets, reis |
| Noodbuffer | Direct beschikbaar spaargeld voor onverwachte uitgaven |
| Jaarlijkse kosten pot | Apart zetten voor kosten die 1x per jaar komen (vakantie, belasting, onderhoud) |
| Onderhoudspot | Voor herstel of vervanging van duurzame goederen |
| Kindpot | Voor onverwachte kosten rondom kinderen (school, zorg, activiteiten) |

### Aanbevolen buffergrootte (richtlijn)

| Huishoudtype | Minimale noodbuffer |
|---|---|
| Alleenstaande | 3 maanden netto inkomen |
| Eenoudergezin | 3-6 maanden netto inkomen |
| Stel | 3 maanden gecombineerd netto inkomen |
| Tweeoudergezin | 3-6 maanden gecombineerd netto inkomen |

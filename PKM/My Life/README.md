# My Life

This is the part of the vault that holds the five life concepts: Topics, Habits, Goals, Projects, and Key Elements. Each has its own flat folder. Everything connects through `[[wikilinks]]` rather than nested hierarchies.

A Topic might link to a Habit. A Habit feeds a Goal. A Goal sits inside a Key Element. A Project is the time-bound push that moves a Goal forward. The folder structure is flat on purpose — the links carry the meaning.

## Five concepts

| Folder | File prefix | Purpose |
|---|---|---|
| `Goals/` | — | What I want to achieve |
| `Habits/` | — | Recurring behaviors that support my Goals |
| `Topics/` | `T - ` | Areas of interest or knowledge (e.g. `T - AI.md`) |
| `Projects/` | `P - ` | Time-bound initiatives that move Goals forward |
| `Key Elements/` | `KE - ` | Core life domains (e.g. `KE - FINANCE.md`, `KE - SELF.md`) |

See `PKM/INDEX.md` for the full file listing.

# Team Inbox

Where you drop raw inputs for the team to process.

Anything you throw in here gets filed. Screenshots, voice memos, business cards, PDFs, half-formed thoughts in a `.md` file, links to articles — drop it in, then say "Larry, process the inbox."

## Structure

```
Team Inbox/
├── Personal/           (Sienna processes, Penn handles journal/narrative)
├── Kamer E-commerce/   (Sasha processes)
└── Geldstroom Regie/   (Finn processes)
```

## How it works

1. Drop a file (or paste content into a `.md` file) in the right domain folder
2. **Larry** routes it to the right specialist
3. The specialist files it into PKM, CRM, Journal, or wherever it belongs
4. Cross-links via `[[wikilinks]]` get added where relevant
5. The processed input gets removed from Team Inbox once filed

## What goes here

- Screenshots, business cards, whiteboard photos
- Voice memos to be transcribed
- Random ideas to capture but not file yourself
- Links for **Pax** to research deeper
- Quick `.md` braindumps for **Penn** to file across PKM

If you know exactly where a note belongs, write it directly there. Team Inbox is for *"I have something, not sure where — just take it."*
